# Gitspace - 代码随香录 (Code Learning Record)

Welcome to Gitspace! This repository serves as a learning space for Git operations and collaborative coding practices.

## 📖 About

Gitspace is a practice repository designed to help developers learn and master Git workflows, branching strategies, and collaborative development techniques.

## 🌿 Repository Purpose

This space is dedicated to:
- Learning Git fundamentals and advanced features
- Practicing branching and merging strategies
- Experimenting with different Git workflows
- Sharing code learning experiences
- Recording coding thoughts and practices

## 🚀 Getting Started

### Prerequisites
- Git installed on your system
- Basic understanding of version control concepts

### Clone the Repository
```bash
git clone https://github.com/Trinjq/Gitspace.git
cd Gitspace
```

## 📚 Learning Resources

### Basic Git Commands
```bash
# Check repository status
git status

# Create a new branch
git branch <branch-name>

# Switch to a branch
git checkout <branch-name>

# Commit changes
git add .
git commit -m "Your commit message"

# Push changes
git push origin <branch-name>
```

### Branching Strategy
This repository uses multiple branches for different purposes:
- `master` - Main stable branch
- `hot-fix` - Quick fixes and patches
- Feature branches - For new developments
